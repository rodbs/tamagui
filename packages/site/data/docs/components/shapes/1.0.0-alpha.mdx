## Usage

Tamagui supports sizing shapes using your `size` tokens, or plain numbers.

```jsx
import { Square, Circle } from 'tamagui'

export default () => (
  <>
    <Square size="$4" />
    <Square size={100} />
    <Circle size="$4" />
    <Circle size={100} />
  </>
)
```

### Shape props

Buttons extend Stack views inheriting all the [Tamagui standard props](/docs/intro/props), plus:

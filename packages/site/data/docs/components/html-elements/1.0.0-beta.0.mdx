# HTML Elements

<Description>Render semantic HTML with these elements.</Description>

To assist in creating accessible web apps, the following components are exported which map to their respective HTML dom elements:

- `Section` (html `section`)
- `Article` (html `article`)
- `Main` (html `main`)
- `Header` (html `header`)
- `Aside` (html `aside`)
- `Footer` (html `footer`)
- `Nav` (html `nav`)

On native, these elements will render the same.

## A

The `<A />` component extends (SizableText)[/docs/components/text#sizable-text], adding the `href`, `target`, and `rel` attributes.

On native, it will use React Native `Linking.openURL`, on web it will render to an `a` element with `href` set appropriately.

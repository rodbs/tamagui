---
title: create-tamagui-app
description: Get started with a helpful template
---

Set up a starter repo with expo, webpack, and tamagui configured and ready to go:

```bash
npx create-tamagui-app@latest my-app
```

Check out [the source of the starter app template](https://github.com/tamagui/starter-base).

To run the app:

```bash
cd my-app
yarn
yarn dev
```
